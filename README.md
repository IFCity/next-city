## Getting Started

First, run the development server:

```bash
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tests

```bash
yarn test
```

## Deploy

Check https://next-city.vercel.app/ 

It deploys automatically when master branch changes
